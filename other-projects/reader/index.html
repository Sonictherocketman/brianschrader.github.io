About this Project

The BiteofanApple Mastodon Client is a toy app built in an evening as a way of playing with the Mastodon API and live streaming.

It stores no data remotely. All keys are kept on device in Local Storage.

Check out the code →

Yes, it does stream data. Just be patient.
